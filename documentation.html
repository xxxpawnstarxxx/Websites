SimpleLibary

SimpleLibary is a UI library designed for creating interactive interfaces for Roblox, it provides a simple yet powerful way to build GUI elements programmatically.

Installation

To use SimpleLibary, paste this first in your executor:

// Load RayJS from GitHub
let script = document.createElement('script');
script.src = 'https://yourusername.github.io/rayjs/RayJS.js';
script.onload = function() {
  // Initialize RayJS after loading
  window.RayJS = LoadRayJS({
    title: 'My Interface',
    theme: 'dark', // or 'light'
    accentColor: '#3366ff'
  });
};
document.head.appendChild(script);

Basic Usage

After loading the library, you can create and add modules to the very bottem

// Create a window
const window = RayJS.NewWindow({
  name: 'My Window',
  size: { width: 400, height: 500 }
});

// Add a button
window.Button({
  Text: 'Click Me',
  Callback: function() {
    console.log('Button clicked!');
    RayJS.Notify({
      title: 'Notification',
      content: 'You clicked the button!'
    });
  }
});

Window Methods

NewWindow(options)
Creates a new window with the specified options.
Parameters:
Name Type Description Default
name string The title of the window 'New Window'
size object Width and height of the window {width: number, height: number} {width: 400, height: 500}
Example
const window = RayJS.NewWindow({
  name: 'Control Panel',
  size: { width: 450, height: 600 }
});

Window Instance Methods

NewTab(name)
Creates a new tab in the window with the specified name.
Parameters:
Name Type Description
name string The name of the tab
Example
const tab = window.NewTab('Settings');

UI Elements

Button

Button(options)
Creates a clickable button element.
Parameters:
Name Type Description Default
Text string The text displayed on the button 'Button'
Callback function Function to execute when button is clicked null
Returns: An object with methods to update or destroy the button.
Method Description
Update(newOptions) Updates the button properties
Destroy() Removes the button from the UI
Example
const button = window.Button({
  Text: 'Execute Script',
  Callback: function() {
    // Load and execute a script from pastebin
    RayJS.LoadScript('https://pastebin.com/raw/example123', function() {
      console.log('Script loaded!');
    });
  }
});

Toggle

Toggle(options)
Creates a toggle switch element.
Parameters:
Name Type Description Default
Text string The label for the toggle 'Toggle'
Default boolean Initial state of the toggle false
Callback function Function to execute when toggle state changes null
Returns: An object with methods to interact with the toggle.
Method Description
Value() Returns current toggle state
Set(value) Sets the toggle state
Destroy() Removes the toggle from the UI
Example
const toggle = window.Toggle({
  Text: 'Enable Feature',
  Default: true,
  Callback: function(state) {
    console.log('Feature is now', state ? 'enabled' : 'disabled');
  }
});

Complete Example

Here's a complete example that demonstrates how to create a full-featured interface with RayJS:

// Load RayJS
let script = document.createElement('script');
script.src = 'https://yourusername.github.io/rayjs/RayJS.js';
script.onload = function() {
  // Initialize RayJS
  window.RayJS = LoadRayJS({
    title: 'Script Hub',
    theme: 'dark',
    accentColor: '#ff3366'
  });

  // Create main window
  const window = RayJS.NewWindow({
    name: 'Script Hub',
    size: { width: 450, height: 600 }
  });

  // Add tabs
  const scriptsTab = window.NewTab('Scripts');
  const settingsTab = window.NewTab('Settings');

  // Scripts tab content
  window.Label({
    Text: 'Script Hub',
    Size: '20px',
    Bold: true,
    Center: true
  });

  window.Line();

  window.Button({
    Text: 'Load Auto-Farm Script',
    Callback: function() {
      RayJS.LoadScript('https://pastebin.com/raw/autofarm123', function() {
        RayJS.Notify({
          title: 'Success',
          content: 'Auto-Farm script loaded!'
        });
      });
    }
  });

  window.Button({
    Text: 'Load ESP Script',
    Callback: function() {
      RayJS.LoadScript('https://pastebin.com/raw/esp456', function() {
        RayJS.Notify({
          title: 'Success',
          content: 'ESP script loaded!'
        });
      });
    }
  });

  window.Space(20);

  window.Textbox({
    Text: 'Load Custom Script',
    Placeholder: 'Pastebin URL...',
    ButtonText: 'Load',
    Callback: function(url) {
      RayJS.LoadScript(url, function() {
        RayJS.Notify({
          title: 'Success',
          content: 'Custom script loaded from: ' + url
        });
      });
    }
  });

  // Settings tab content
  settingsTab.button.click(); // Switch to settings tab

  window.Label({
    Text: 'Settings',
    Size: '20px',
    Bold: true,
    Center: true
  });

  window.Line();

  const themeToggle = window.Toggle({
    Text: 'Dark Mode',
    Default: true,
    Callback: function(state) {
      RayJS.Notify({
        title: 'Theme Changed',
        content: state ? 'Dark mode enabled' : 'Light mode enabled'
      });
    }
  });

  window.ColorPicker({
    Text: 'Accent Color',
    Default: '#ff3366',
    Callback: function(color) {
      RayJS.Notify({
        title: 'Color Changed',
        content: 'Accent color updated to: ' + color
      });
    }
  });

  window.Keybind({
    Text: 'Toggle Interface',
    Default: 'F1',
    Callback: function(key) {
      RayJS.Notify({
        title: 'Keybind Set',
        content: 'Press ' + key + ' to toggle the interface'
      });
    }
  });

  // Switch back to scripts tab
  scriptsTab.button.click();
};
document.head.appendChild(script);

This example creates a script hub interface with two tabs (Scripts and Settings), various interactive elements, and demonstrates how to load external scripts from Pastebin URLs.
